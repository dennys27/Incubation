


# Incubation

where users can book slots.

  User  
1.user signup and login  
2.application


Admin

slot allocation   
Notification with toast

mongoose

## Run Locally

Clone the project

```bash
https://github.com/dennys27/Incubation.git
```

Go to the project directory

```bash
  cd incubation
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
